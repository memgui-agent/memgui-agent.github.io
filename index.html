Long-horizon mobile GUI agents

MemGUI-Agent

An end-to-end mobile GUI agent with proactive context management. MemGUI-Agent instantiates ConAct, a Context-as-Action paradigm that lets the policy fold history, update UI memory, and act in one structured response.

Method

ConAct: Context-as-Action

Instead of carrying an ever-growing raw transcript, MemGUI-Agent makes context updates explicit and executable inside each model response.

Folded Action History

Compresses completed interaction spans while preserving task-relevant progress.

Folded UI State

Stores persistent UI facts such as names, prices, values, and form state.

Recent Step Record

Keeps local interaction continuity for the next GUI action.

Release

Code, data, model, and benchmarks

Code Public MemGUI-Agent release repository. MemGUI-3K 2,956 trajectories and multimodal training data. MemGUI-8B-SFT 8B model trained with LoRA SFT on MemGUI-3K. MobileWorld OOD mobile GUI benchmark integration.
